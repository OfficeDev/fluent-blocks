# Type scale

Fluent and the UI Kit specify a series of type sizes also paired with font weights which signify where in the structure of the page a heading and the following content belongs.

- [Fluent › Text](https://www.figma.com/file/IkBEp9pnO4Ep7HH4Bq5xrW/Text?node-id=2%3A895)
- [UI Kit › Design system › Type](https://www.figma.com/file/UwN095fVIvwZDNtzTtUE3E/Microsoft-Teams-UI-Kit-Internal?node-id=4303%3A180379)

The `Heading` component maps 4 of these size-weight pairs to semantic HTML tags. A `Heading` element’s place in the document structure is managed automatically by the `Section` component.

All `Heading`s apply a top margin of 24px and the same bottom margin as `Paragraph` of 4px.
